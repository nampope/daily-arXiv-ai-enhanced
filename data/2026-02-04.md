### [1] [Grid Diagrams of Fibered Knots](https://arxiv.org/abs/2602.02642)
*Paul Leon Itzlinger*

Main category: math.GT

TL;DR: 研究网格图中是否存在具有唯一最大Alexander分级网格状态的网格图，用于检测纤维结，并开发算法在5397个纤维结中找到了5385个符合条件的网格图。


<details>
  <summary>Details</summary>
Motivation: Ghiggini和Ni证明了结Floer同调可以检测纤维结，这意味着具有唯一最大Alexander分级网格状态的网格图只存在于纤维结中。但问题是：是否每个纤维结都允许这样的特殊网格图？这是本文研究的核心动机。

Method: 开发了一种高效的方法来判断给定网格图是否满足更严格的条件：具有唯一实现Alexander函数上界的网格状态。将此方法实现为Python包，对交叉数不超过13的5397个纤维素结进行系统搜索。

Result: 在5397个交叉数不超过13的纤维素结中，找到了5385个符合条件的网格图，覆盖了绝大多数纤维结，表明这种特殊网格图在纤维结中广泛存在。

Conclusion: 研究结果表明，具有唯一最大Alexander分级网格状态的网格图在纤维结中普遍存在，为Ghiggini和Ni的定理提供了大量实证支持，但仍有少数纤维结未找到这样的网格图，留下了进一步研究的空间。

Abstract: Grid diagrams are special representations of knots in the three-sphere that are used to define a combinatorial version of knot Floer homology. Paolo Ghiggini and Yi Ni showed that knot Floer homology detects fibered knots. Their results imply, in particular, that grid diagrams with a unique grid state whose Alexander grading is maximal only exist for fibered knots. Whether every fibered knot admits such a diagram remains an open question. Here, we investigate the existence of such special grid diagrams for fibered knots. We develop an efficient method for deciding whether a given grid diagram meets the even stricter condition of having a unique grid state that realizes an upper bound for the Alexander function. By implementing this method in a Python package, we find suitable grid diagrams for 5385 of the 5397 fibered prime knots with crossing number at most 13.

### [2] [A U-match Algorithm for Persistent Relative Homology](https://arxiv.org/abs/2602.03163)
*Christian Lentz,Gregory Henselman-Petrusek,Lori Ziegelmeier*

Main category: math.AT

TL;DR: 本文提出了一种计算持久相对同调（PRH）的简单透明方法，使用两步矩阵约简技术，时间复杂度与普通持久同调相当。


<details>
  <summary>Details</summary>
Motivation: 数据驱动科学中需要解释高维噪声数据的结构，拓扑数据分析（TDA）通过持久同调提供了解决方案，但需要扩展到相对子空间的拓扑结构分析。

Method: 提出计算持久相对同调（PRH）的两步矩阵约简技术，该方法简单、透明且通用，最坏情况时间复杂度与普通持久同调相当。

Result: 证明了该方法的正确性和计算复杂度，并为特殊情况提供了性能优化的实现。

Conclusion: 持久相对同调框架扩展了拓扑数据分析能力，提供了一种高效计算相对拓扑结构的方法，具有实际应用价值。

Abstract: A central problem in data-driven scientific inquiry is how to interpret structure in noisy, high-dimensional data. Topological data analysis (TDA) provides a solution via persistent homology, which encodes features of interest as topological holes within a filtration of data. The present work extends this framework to a related invariant which uncovers topological structure of a space relative to a subspace: persistent relative homology (PRH). We show that this invariant can be computed in a simple, highly transparent and general manner, using a two-step matrix reduction technique with worst-case time complexity comparable to ordinary persistent homology. We provide proofs demonstrating the correctness and computational complexity of this approach in addition to a performance-optimized implementation for a special case.

### [3] [On the maximal perimeter of isotropic log-concave probability measures](https://arxiv.org/abs/2602.03831)
*Silouanos Brazitikos,Apostolos Giannopoulos,Antonios Hmadi,Natalia Tziotziou*

Main category: math.MG

TL;DR: 研究各向同性对数凹概率测度的最大周长常数Γ_n，证明了Γ_n ≤ Cn^{3/2}的上界，改进了之前的O(n^2)结果，并在附加结构假设下得到了最优的线性界O(n)。


<details>
  <summary>Details</summary>
Motivation: 研究各向同性对数凹概率测度的最大周长常数Γ(μ)，该常数衡量了凸集相对于测度μ的最大可能边界贡献。目标是确定Γ_n = sup{Γ(μ)}的增长阶数，这对于理解高维对数凹测度的几何性质具有重要意义。

Method: 通过分析各向同性对数凹概率测度的几何性质，采用概率和凸几何方法，建立了Γ_n的上界估计。在附加结构假设下，进一步优化了分析方法以获得更紧的线性界。

Result: 证明了Γ_n ≤ Cn^{3/2}，其中C>0是绝对常数，这改进了之前已知的O(n^2)上界。在附加结构假设下，得到了最优的线性界O(n)。

Conclusion: 各向同性对数凹概率测度的最大周长常数Γ_n的增长阶数最多为n^{3/2}，显著改进了之前的估计。在某些结构条件下，该界可以进一步优化到线性阶，揭示了这类测度边界行为的精细结构。

Abstract: We study the maximal perimeter constant of isotropic log-concave probability measures on $\mathbb{R}^n$. For a measure $μ$, this quantity, denoted by $Γ(μ)$, is defined as the supremum of the $μ$-perimeter over all convex bodies and measures the largest possible boundary contribution of convex sets with respect to $μ$. Let $$Γ_n := \sup\{Γ(μ) : μ\text{ is an isotropic log-concave probability measure on } \mathbb{R}^n\}.$$ We prove that $Γ_n \leqslant Cn^{3/2}$, where $C>0$ is an absolute constant. This result improves the previously known $O(n^2)$ upper bound. Under additional structural assumptions, we obtain sharp linear bounds of order $O(n)$.

</details>
